Nous voulons avoir une nouvelle classe Item telle que :
Un nouvel article peut être créé avec 
Item(price, weight)
On aura accès au prix et au poids d'un article avec Item.price et Item.weight.
Ecrivez le code de cette classe, avec le constructeur qui convient.
Exemple de code utilisant la classe : 
item = Item(10, 5)
print(item.price) # 10
print(item.weight) # 5
